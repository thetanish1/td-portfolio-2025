Weather Dashboard

A weather dashboard that displays current weather conditions and 5-day forecast for any city worldwide. The app uses the OpenWeatherMap API to fetch weather data and stores search history in local storage.

Key Features:

  • Current weather display with icons
  • 5-day forecast with temperature, humidity, and wind speed
  • Search history with click-to-reload functionality
  • Responsive design that works on all devices
  • UV index display with color-coded severity

Technologies Used: JavaScript, jQuery, Bootstrap, OpenWeatherMap API
